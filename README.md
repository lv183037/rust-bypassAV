## 个人学习过程记录
### 使用方法


1. 安装`rust`环境 具体安装方法参考 `https://course.rs/first-try/installation.html`
2. `cobalt strike` 生成 `raw` 格式的 `bin` 文件
3. 生成的`beacon.bin`和`icon.ico` 放在同目录下 直接运行`rust-bypassAV.exe`即可
